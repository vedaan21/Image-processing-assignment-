# Intelligent Image Processing System

## Objective

To build an end-to-end image processing system including enhancement, restoration, segmentation, and feature extraction.

---

## Tools Used

* Python
* OpenCV
* NumPy
* Matplotlib

---

## Steps Performed

1. Image preprocessing
2. Noise addition and filtering
3. Enhancement (Histogram Equalization)
4. Segmentation (Global + Otsu)
5. Edge detection & contours
6. Feature extraction (ORB)
7. Performance evaluation (MSE, PSNR, SSIM)

---

## Outputs

* Enhanced images
* Segmented images
* Feature extraction results
* Full pipeline visualization

(All saved in `outputs/` folder)

---

## Observations

* Noise reduces quality
* Filters restore images
* Otsu gives better segmentation
* ORB detects important features

---

## How to Run

```bash
python3 main.py
```
