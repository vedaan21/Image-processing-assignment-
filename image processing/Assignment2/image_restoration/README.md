# Image Restoration using Python

## Objective
To simulate noise in surveillance images and restore quality using filters.

## Noise Types
- Gaussian Noise
- Salt & Pepper Noise

## Filters Used
- Mean Filter
- Median Filter
- Gaussian Filter

## Metrics
- MSE
- PSNR

## Observations
- Median filter best for salt-pepper noise
- Gaussian filter best for Gaussian noise
